## Quick start
1. Клонировать репо:
   ```bash
   git clone https://github.com/nuradevo/dms-showcase.git
   cd dms-showcase
   ```
2. Запустить backend (пример):
   ```bash
   # примерная команда, адаптируйте под ваш Makefile / docker-compose
   make up
   ```
3. Открыть http://localhost:3000
