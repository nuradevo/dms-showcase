## Как тестировать
1. Склонировать ветку
2. Запустить `make up`
3. Выполнить `make seed` и открыть frontend/backend

## Примечания
- В конфигах используются примерные значения для .env. Перед публикацией замените JWT_SECRET и другие секреты.
